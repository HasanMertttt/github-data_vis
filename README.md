## 📌 Project Overview
This project demonstrates an end-to-end data analytics workflow on a synthetic coffee shop chain dataset. It serves as a foundational baseline for autonomous cafe analytics and sales forecasting systems. The workflow covers everything from raw data generation and structural preprocessing to advanced exploratory data analysis (EDA) with a dark-themed, neon aesthetic.

## 🛠️ Technical Workflow
1. **Data Generation (`coffee_shop_dataset_generator.ipynb`):** Built a synthetic dataset (~2000 records) simulating real-world coffee shop transactions, including timestamps, branch locations, and customer demographics.
2. **Data Cleaning & Preprocessing (`coffee_shop_data_cleaning.ipynb`):** 
   - Handled deliberately injected missing values and outliers.
   - Formatted complex datetime columns for time-series compatibility.
   - Structured the data for high-performance querying.
3. **Advanced EDA & Business Intelligence (`coffee_shop_eda.ipynb`):**
   - **Revenue Analysis:** Branch-by-branch total revenue comparisons.
   - **Time Series Trend:** Analyzed daily revenue fluctuations and established a 7-day moving average (trend line) essential for future forecasting models.
   - **Feature Correlation:** Developed a heatmap to identify relationships between quantitative variables (e.g., spending amount, quantity, and customer age).

## 📊 Visual Aesthetics
All visualizations were custom-designed using a dark mode aesthetic with a neon purple and black color palette, ensuring that the Business Intelligence (BI) insights are not only accurate but also visually striking and ready for modern dashboard integration.

## 🚀 Key Business Insights
- Established a clean data pipeline ready for Machine Learning (Forecasting) integration.
- Identified clear weekly sales trends through moving average smoothing.
- Verified expected correlations (e.g., strong positive correlation between quantity and total transaction amount) to validate dataset integrity.
